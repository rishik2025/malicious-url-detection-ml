# malicious-url-detection-ml
Machine Learning based system to detect malicious URLs using feature extraction and VirusTotal API for real-time threat intelligence.
## Project Overview
This project detects malicious URLs using machine learning and VirusTotal API.

## Features
- URL scanning
- File hash scanning
- QR code scanning
- Report generation

## Technologies Used
- Python
- Scikit-learn
- Pandas
- VirusTotal API
